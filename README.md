This is a ReactJS project bootstrapped with [Vite] and built to manage personal tasks with To Do, In Progress, and Done columns, user authentication, and user profile management.

## Features

User Registration & Login

Create new users and log in to access personal tasks.

Stores token and userName in localStorage for session management.

Task Management

View tasks categorized into To Do, In Progress, and Done.

Add, edit, and delete tasks.

Columns are styled with borders and colored cards for better UI separation.

User Profile

Create or update user profile with age, bio, and location.

Form validation using Formik and Yup.

Responsive UI

Fully responsive layout using MUI Grid and Cards.

Each task column has a styled border and padding.

API Integration

REST API integration using Axios.

Frontend handles task fetching, creation, updating, and deletion.

Error Handling

Shows alerts for errors in login, registration, profile update, and task operations.

React Query

Efficient data fetching and caching for tasks.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
```

Open http://localhost:5173
in your browser to see the result (Vite default port is 5173).

## Dependencies

React 18

Vite – Frontend build tool

React Router – Routing

Axios – HTTP requests

React Query – Data fetching and caching

Material UI – UI components

Formik & Yup – Form handling and validation
